# Kameoka Lab Website

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Decap CMS

Open `/admin` on the deployed site. Update `public/admin/config.yml` with your Vercel OAuth base URL.

OAuth base URL example: `https://xxxxx.vercel.app`

## Researchmap import

```bash
python scripts/convert_researchmap.py
```
